
# Оно тебе надо

Одностраничный сайт аукциона необычных вещей, выполненный в виде статической HTML/CSS-вёрстки с точным соответствием дизайн-макету.

## О проекте

Проект представляет собой лендинг аукциона, на котором представлены нестандартные лоты: книги, фильмы и картины, не принятые массовой культурой, но ставшие культовыми со временем.

Цель проекта — продемонстрировать навыки:
- точной вёрстки по дизайн-макету,
- работы с CSS Grid и Flexbox,
- аккуратной типографики и отступов,
- построения семантической HTML-структуры,
- соблюдения методологии БЭМ.

Проект выполнен без использования JavaScript — фокус сделан исключительно на качестве вёрстки и визуальном соответствии макету.

## Используемые технологии

- HTML5 (семантическая разметка)
- CSS3  
  - Grid Layout  
  - Flexbox  
- Методология БЭМ 
- Pixel Perfect
- Git, GitHub
